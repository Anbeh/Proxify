# Proxify: Get Latest Proxies Instantly

Proxify is a repository for accessing the latest proxies and V2Ray configurations, all kept up-to-date automatically.

## Supported protocols

- SOCKS5
- SOCKS4
- HTTP
- HTTPS
- MTProto / SOCKS For Telegram messenger
